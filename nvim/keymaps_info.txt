================================================================
                    Neovim 키맵 목록
================================================================
  Leader 키: \  (백슬래시)
================================================================

[ 창 이동 ]  (노멀 모드)

  Ctrl + h       왼쪽 창으로 이동
  Ctrl + l       오른쪽 창으로 이동
  Ctrl + j       아래 창으로 이동
  Ctrl + k       위 창으로 이동

----------------------------------------------------------------
[ 창 크기 조절 ]  (노멀 모드)

  Ctrl + Up      창 높이 증가
  Ctrl + Down    창 높이 감소
  Ctrl + Left    창 너비 감소
  Ctrl + Right   창 너비 증가

----------------------------------------------------------------
[ 줄 편집 ]  (비주얼 모드)

  <              들여쓰기 감소 (선택 유지)
  >              들여쓰기 증가 (선택 유지)
  J              선택한 줄 아래로 이동
  K              선택한 줄 위로 이동

----------------------------------------------------------------
[ 기본 단축키 ]  (노멀 모드)

  Esc            검색 강조 표시 해제
  \ w            파일 저장
  \ q            nvim 종료
  F5             Python 저장 후 실행 (터미널 분할, 종료 시 자동 닫기)

----------------------------------------------------------------
[ 파일 탐색기 - nvim-tree ]  (노멀 모드)

  \ e            파일 탐색기 열기 / 닫기 토글
  \ ef           현재 파일을 탐색기에서 위치 찾기
  \ ec           탐색기 디렉토리 전체 접기

----------------------------------------------------------------
[ 퍼지 파인더 - Telescope ]  (노멀 모드)

  \ ff           파일 이름으로 찾기
  \ fg           전체 텍스트 검색 (grep)
  \ fb           열린 버퍼 목록 보기
  \ fh           도움말 태그 검색
  \ fr           최근에 열었던 파일 목록
  \ fc           컬러 스킴 변경

----------------------------------------------------------------
[ 버퍼 탭 - bufferline ]  (노멀 모드)

  Tab            다음 버퍼로 이동
  Shift + Tab    이전 버퍼로 이동
  \ x            현재 버퍼 닫기
  \ bo           현재 버퍼 외 나머지 모두 닫기

----------------------------------------------------------------
[ 자동완성 - nvim-cmp ]  (입력 모드 / 스니펫 모드)

  Ctrl + Space   자동완성 목록 강제 열기
  Ctrl + e       자동완성 목록 닫기
  Enter          선택한 항목으로 자동완성 확정
  Tab            다음 항목 선택 / 스니펫 다음 위치로 이동
  Shift + Tab    이전 항목 선택 / 스니펫 이전 위치로 이동
  Ctrl + b       문서 미리보기 위로 스크롤
  Ctrl + f       문서 미리보기 아래로 스크롤

----------------------------------------------------------------
[ AI 어시스턴트 - Claude (codecompanion) ]

  노멀 / 비주얼 모드
  \ ac           Claude 채팅 창 열기 / 닫기 (토글)
  \ aa           액션 팔레트  (코드 설명·리뷰·리팩토링 등)
  \ ai           인라인 어시스트  (커서 위치에 직접 수정)

  비주얼 모드
  \ as           선택 영역을 채팅에 추가
  Ctrl + A       선택 영역을 채팅에 전송

----------------------------------------------------------------
[ 터미널 / 기타 ]

  \ t            내장 터미널 열기  (노멀 모드)
  Esc            터미널 → 노멀 모드 전환  (터미널 모드)
  \ oc           init.lua 설정 파일 빠르게 열기  (노멀 모드)
