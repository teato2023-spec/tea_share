● \ac 채팅 창 사용법                                                                                                                   
                                                                                                                                       
  기본 조작                                                                                                                            
   
  ┌─────────────────┬──────────────────────────────┐                                        
  │       키        │             동작                                                                                        
  │ i               │ 입력 모드 진입 (메시지 작성) 
  │ Enter (입력 중) │ 줄바꿈                       
  │ Ctrl+s          │ 메시지 전송 (입력 모드에서)  
  │ Esc             │ 노멀 모드로 전환             
  │ q               │ 채팅 창 닫기 (노멀 모드에서) 
  │ \ac             │ 채팅 창 토글 (열기/닫기)      └─────────────────┴──────────────────────────────┘                                                                                                                                          
  코드와 함께 사용하는 방법                                                                                                          
                                                                                                                                     
  ① 코드 선택 후 채팅에 전달                                                                                                         
  비주얼 모드로 코드 선택 → \as  (채팅창에 코드 붙여넣기)                                                                              
  → 채팅창에서 i → 질문 입력 → Ctrl+s                                                                                                
                                                                                                                                       
  ② 액션 팔레트로 빠르게                                                                                                               
  코드 선택 → \aa                                                                                                                      
  메뉴에서 선택:                                                                                                                       
  - Explain Code — 코드 설명                                                                                                           
  - Fix Code — 버그 수정                                                                                                               
  - Generate Tests — 테스트 코드 생성                                                                                                  
  - Refactor Code — 리팩토링                                                                                                           
                                                                                                                                     
  ③ 인라인 수정 (가장 편함)                                                                                                            
  코드 선택 → \ai → "이 함수를 더 간결하게 수정해줘" → Enter                                                                           
  → 현재 파일에 직접 수정 결과가 삽입됨                                                                                                
                                                                                                                                       
  ---                                                                                                                                  
  채팅창 안에서 슬래시 명령어                                                                                                          
                                                                                                                                       
  입력 모드에서 / 입력하면 명령어 목록이 뜸:                                                                                         
                                                                                                                                                                                                                 
  │ 명령어  │                    기능                     │                                                                                           │ /buffer │ 현재 열린 버퍼(파일) 내용을 컨텍스트로 추가 │                                                                          
  │ /file   │ 특정 파일을 컨텍스트로 추가                 │                                                                            
  │ /help   │ nvim 도움말을 컨텍스트로 추가               │                                                                            
  │ /clear  │ 채팅 기록 초기화                            
 ---                                                                                                                                  
  실제 사용 예시                                                                                                                     
                                                                                                                                       
  1. Python 파일 열기                                                                                                                
  2. 함수 전체 선택 (V로 줄 선택)                                                                                                      
  3. \as  → 채팅창에 코드 전송                                                                                                         
  4. i 눌러서 입력 모드                                                                                                                
  5. "이 코드에서 버그를 찾아줘" 입력                                                                                                  
  6. Ctrl+s 로 전송                                                                                                                    
  7. Claude 답변 확인
