----------------------------------------------------------------
  [1] 기본 (플러그인 없음)  ·  config/keymaps.lua
----------------------------------------------------------------

  < 창 이동 >  노멀 모드
    Ctrl + h          왼쪽 창으로 이동
    Ctrl + l          오른쪽 창으로 이동
    Ctrl + j          아래 창으로 이동
    Ctrl + k          위 창으로 이동

  < 창 크기 조절 >  노멀 모드
    Ctrl + ↑          창 높이 증가
    Ctrl + ↓          창 높이 감소
    Ctrl + ←          창 너비 감소
    Ctrl + →          창 너비 증가

  < 줄 편집 >  비주얼 모드
    <                 들여쓰기 감소 (선택 유지)
    >                 들여쓰기 증가 (선택 유지)
    J                 선택한 줄 아래로 이동
    K                 선택한 줄 위로 이동

  < 기타 >  노멀 모드
    Esc               검색 강조 해제
    \ w               파일 저장
    \ q               종료
    F5                Python 저장 후 실행
                      (하단에 터미널 분할 → 종료 시 자동 닫힘)
                      ※ 일반 파일(.py)에서만 동작


----------------------------------------------------------------
  [2] nvim-tree  ·  keymaps/nvim-tree.lua
----------------------------------------------------------------

  노멀 모드
    \ e               파일 탐색기 열기 / 닫기 (토글)
    \ ef              현재 열린 파일을 탐색기에서 위치 찾기
    \ ec              탐색기 디렉토리 전체 접기

  ※ 탐색기 내부 단축키 (nvim-tree 기본값)
    Enter / o         파일 열기 또는 폴더 펼치기
    a                 새 파일 / 폴더 생성
    d                 삭제
    r                 이름 변경
    x                 잘라내기
    c                 복사
    p                 붙여넣기
    q                 탐색기 닫기
    g?                도움말 보기


----------------------------------------------------------------
  [3] Telescope  ·  keymaps/telescope.lua
----------------------------------------------------------------

  노멀 모드
    \ ff              파일 이름으로 찾기
    \ fg              전체 텍스트 검색 (live grep)
    \ fb              열린 버퍼 목록 보기
    \ fh              도움말 태그 검색
    \ fr              최근에 열었던 파일 목록
    \ fc              컬러 스킴 변경

  ※ Telescope 창 내부 단축키 (기본값)
    Ctrl + j / ↓      다음 항목 선택
    Ctrl + k / ↑      이전 항목 선택
    Enter             선택한 항목 열기
    Ctrl + x          수평 분할로 열기
    Ctrl + v          수직 분할로 열기
    Ctrl + t          새 탭으로 열기
    Esc               Telescope 닫기

----------------------------------------------------------------
  [8] AI 어시스턴트 - Claude  ·  keymaps/codecompanion.lua
----------------------------------------------------------------

  노멀 / 비주얼 모드
    \ ac              Claude 채팅 창 열기 / 닫기 (토글)
    \ aa              액션 팔레트
                      (코드 설명 / 리뷰 / 리팩토링 / 버그 찾기 등)
    \ ai              인라인 어시스트
                      (커서 위치 또는 선택 범위를 직접 수정)

  비주얼 모드
    \ as              선택 영역을 채팅 창에 추가
    Ctrl + A          선택 영역을 채팅 창에 전송

  ※ 사용 방법
    · 코드 설명  : 코드 선택 → \ aa → "Explain" 선택
    · 버그 수정  : 코드 선택 → \ ai → 지시사항 입력
    · 자유 채팅  : \ ac → 채팅 창에서 질문
    · API 키     : ~/.bashrc 의 ANTHROPIC_API_KEY 에 설정
